# FinishMyBookAI

An AI-powered tool to help complete unfinished stories and manuscripts.

## Features

- **Upload or Paste** - Support for .txt, .md, .doc files or direct text input
- **Genre Detection** - Auto-detects or manually select:
  - Fantasy
  - Science Fiction
  - Romance
  - Mystery/Thriller
  - Horror
  - And more
- **Style Matching** - AI continues writing in your voice
- **Word Count Tracking** - Monitor your story's length

## How to Use

1. Open `index.html` in a browser
2. Upload your unfinished story or paste text directly
3. Select genre (or let AI auto-detect)
4. Choose how much to generate
5. AI continues your narrative

## Why Use This?

- Overcome writer's block
- Get story continuation ideas
- See how AI interprets your style
- Finish abandoned drafts

## Tech Stack

- HTML/CSS/JavaScript frontend
- AI API integration for text generation
- Drag-and-drop file upload

## Getting Started

```bash
# Just open in browser
open index.html
```

No build process required - pure client-side application.
